## 📌 About

A front-end **Amazon.com clone** built entirely with **HTML and CSS**, created as a web development practice project. The goal was to replicate the look and feel of Amazon's layout to sharpen:

- 🎨 Complex CSS layout techniques
- 📱 Responsive design patterns
- 🖼️ Working with images and assets
- 🌐 Real-world e-commerce UI structure
